# Tic-Tac-Toe-Game

To play just download and open the `Tic_Tac_Toe_Game.exe` file.

The original file can also be found at Tic_Tac_Toe_Game > bin > Release > `Tic_Tac_Toe_Game.exe`. The process is the same.
